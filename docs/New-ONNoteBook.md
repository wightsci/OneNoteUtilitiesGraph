# New-ONNoteBook

## SYNOPSIS
Creates a new OneNote NoteBook.

## SYNTAX

```
New-ONNoteBook [-DisplayName] <String> [<CommonParameters>]
```

## DESCRIPTION
Creates a new OneNote NoteBook.

## EXAMPLES
