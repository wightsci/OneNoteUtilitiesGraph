# Get-ONDefaultSection

## SYNOPSIS
Gets the default OneNote Section for new content.

## SYNTAX

```
Get-ONDefaultSection [<CommonParameters>]
```

## DESCRIPTION
Gets the default OneNote Section for new content.

## EXAMPLES

```
Get-ONDefaultSection
```
