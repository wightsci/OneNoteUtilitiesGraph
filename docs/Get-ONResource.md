# Get-ONResource

## SYNOPSIS
Saves a OneNote Resource to a file.

## SYNTAX

```
Get-ONResource -Uri <String> -Path <String> [<CommonParameters>]
```

## DESCRIPTION
This cmdlet retreives a binary resource (imge, file) from OneNote and saves it in a file.
