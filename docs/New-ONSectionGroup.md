# New-ONSectionGroup

## SYNOPSIS
Creates a new OneNote SectionGroup.

## SYNTAX

### ByUri
```
New-ONSectionGroup -DisplayName <String> -Uri <String> [<CommonParameters>]
```

### ByID
```
New-ONSectionGroup -DisplayName <String> -Id <String> [<CommonParameters>]
```

## DESCRIPTION
Creates a new OneNote SectionGroup.

## EXAMPLES

