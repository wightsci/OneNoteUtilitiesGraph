# Get-ONSectionGroup

## SYNOPSIS
Gets a OneNote SectionGroup.

## SYNTAX

### uri
```
Get-ONSectionGroup -Uri <String> [<CommonParameters>]
```

### id
```
Get-ONSectionGroup -Id <String> [<CommonParameters>]
```

## DESCRIPTION
Gets a OneNote SectionGroup using its URI or ID.
