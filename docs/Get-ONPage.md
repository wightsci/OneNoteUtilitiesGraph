# Get-ONPage

## SYNOPSIS
Gets a OneNote Page.

## SYNTAX

### uri
```
Get-ONPage -Uri <String> [<CommonParameters>]
```

### id
```
Get-ONPage -Id <String> [<CommonParameters>]
```

## DESCRIPTION
Gets a OneNote Page.
